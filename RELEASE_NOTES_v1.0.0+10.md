# WanMap v1.0.0+10 リリースノート

## 📅 リリース日
2025-01-16

## ✨ 新機能・改善

### 🎨 UI/UXの大幅改善
1. **TOPページのビジュアル強化**
   - 箱根カード: 3色グラデーション背景 + 装飾アイコン
   - エリアカード: エリア別カラー（横浜・鎌倉・伊豆など）+ グラデーション
   - ルートカード: サムネイルにグラデーション背景
   - 全体的に視覚的に豊かなデザインに改善

2. **箱根観光デジタルマップバナー追加**
   - TOPページ「おすすめエリア」セクションにプロモーションバナー追加
   - タップで箱根観光デジタルマップ（https://map-hakone.staynavi.direct/）へ遷移

3. **TOPページのセクション順序変更**
   - 新しい順序:
     1. 最新のピン投稿（コミュニティの最新情報を最優先）
     2. 今月の人気コース
     3. おすすめエリア
     4. 高評価スポット

### 🗺️ 箱根エリアの改善
1. **箱根エリアの階層化**
   - 箱根を5つのサブエリアに分割（箱根湯本、宮ノ下、強羅、仙石原、芦ノ湖）
   - エリア選択画面で箱根親カードをタップ → サブエリア選択画面へ遷移
   - 各サブエリアに専用のアイコンとカラー

2. **交通アクセス情報の追加**
   - 各箱根サブエリアのルート一覧画面に交通情報カードを表示
   - エリアへの行き方（IC、道路情報など）を明示

3. **エリア説明の改善**
   - DogHub関連の文言を削除し、観光地としての説明に変更
   - エリアの特徴 → 交通情報の順で表示

### 🐛 バグ修正
1. HomeTabの型エラー修正
2. WanMapTypographyのメソッド名修正（h3 → titleLarge）
3. メソッドのスコープ修正

## 🔧 技術的改善
- url_launcherパッケージ追加（外部リンク対応）
- コンポーネントの最適化
